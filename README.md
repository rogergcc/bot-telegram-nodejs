# bot-telegram-nodejs
bot telgram nodejs pokemon. Send number and return pokemon.

  **_bothunter_** in telegram

# Technologies used.
Nodejs,   
body-parser,
dotenv,
express,
node-fet,
telegraf

# Getting Started
1. Clone repo and install npm nodejs git.
2. npm i
3. Run script:

  ```
    npm run dev
  ```
.env
```
DB_NAME=bot_telegram
HOST=http://localhost
PORT=3002
TELEGRAM_API_TOKEN=
WEBHOOK=
```

Send a number of pokemon for the information.
